## How to build

```
$ git clone https://github.com/DietrichPaul/Winkel.git
$ cd Winkel
$ gradlew build
```

In addition to the build-instructions, there are also instructions in
the [Fabric Wiki](https://fabricmc.net/wiki/tutorial:setup).
